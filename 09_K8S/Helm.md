## Install
```
curl -fsSL -o get_helm.sh https://raw.githubusercontent.com/helm/helm/master/scripts/get-helm-3
chmod 700 get_helm.sh
./get_helm.sh
```
## Chart LifeCycle
### Create Chart
```
helm create tchart
# inspect
tree tchart
```
### Reset 
Delete all template files or all files.
```
rm -v  tchart/templates/*
```
Create Chart.yaml
```
apiVersion: v1    
name: tnginx    
version: 0.1.0    
appVersion: 1.0    
    
description: nginx test    
```
