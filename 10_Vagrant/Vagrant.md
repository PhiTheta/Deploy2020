## Install
```
wget https://releases.hashicorp.com/vagrant/2.2.9/vagrant_2.2.9_x86_64.deb
sudo gdebi downloaded_file
```
## Verify
```
vagrant --version
```
## Install disk size plugin
```
vagrant plugin install vagrant-disksize
```
